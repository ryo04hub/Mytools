### 説明
- 自分のインスタグラムから画像を取得して、表示します
- 自動でスクロールします
- 使用する際は、index.htmlの「clientId, userId, accessToken」に、Instagram Developerから発行された値を入力してください

### 使用したもの
- Instagram API
- jQuery
- instafeed.js
